Front-end, Back-end or Full-stack? Development or Engineering?

Yep, this may get confusing. I consider myself falling into one of the following categories: Software Developer/Engineer or Full-stack Web Developer. The reason for that is that I am proficient in technologies allowing me to create fully working web applications - both client and server side code. At the moment my focus is on the web but am also interested in cross-platform technologies such as NativeScript or React Native that allow to create mobile apps for a range of platforms using pretty much the same codebase. And who knows I might be writing games in C# in the future.
